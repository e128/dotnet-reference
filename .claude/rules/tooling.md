# Tooling

**Required CLI tools** (must be installed):
- `rg` (ripgrep) — fast regex search
- `fd` — fast file finder, respects `.gitignore`
- `jq` — JSON processor for script output parsing
- `bash` 5+ — required for associative arrays and modern features

**Optional:**
- `shellcheck` — bash script linter
- `dotnet-outdated-tool` — NuGet package update checker (`dotnet tool install -g dotnet-outdated-tool`)
